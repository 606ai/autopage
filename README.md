# AutoPage - AI-Powered Website Builder

[AutoPage](https://autopage.icu) is a cutting-edge AI-powered website builder that helps you create stunning websites instantly. Our platform automates web design and content creation, making it easier than ever to build your online presence.

## Features

- **AI-Powered Design**: Get intelligent design suggestions based on your content and brand
- **Automated Content Generation**: Generate engaging content with our advanced AI
- **Responsive Design**: Your website looks perfect on all devices
- **SEO Optimization**: Built-in tools to help your website rank higher

## Tech Stack

- Next.js 14
- TypeScript
- React
- Chakra UI
- Tailwind CSS
- React Query
- Zustand

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/606ai/autopage.git
cd autopage
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
